# 📞 مكالمة أحمد

موقع لاستقبال مكالمات صوتية حية مع الوكيل الصوتي "أحمد" (ElevenLabs Conversational AI) — بدون رقم هاتف.

## الاستخدام
- افتح الصفحة، اضغط **اتصل بأحمد**، واسمح للميكروفون.
- تحدّث بشكل طبيعي.

## الرنين التلقائي
الصفحة تفحص `signal.json` كل 5 ثوانٍ. عند تغيّر الحقل `id`، ترنّ الصفحة كمكالمة واردة.
لتشغيل رنة: حدّث `id` في `signal.json` إلى رقم جديد.
